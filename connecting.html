Getting started

Perpex runs on Solana mainnet. Connect any supported wallet, receive a one-time demo balance, and start trading perpetuals in under a minute.

Supported wallets

Perpex supports any wallet that speaks the Solana wallet-standard:

Phantom Solflare MetaMask (Solana Snap)

Connect in four steps

  1. Visit tradeperpex.fun/launchpad
  2. Click Connect wallet in the top-right corner
  3. Approve the connection in your wallet extension
  4. New wallets automatically receive 50 USDC demo balance to explore

Get demo funds

Every new wallet gets a one-time 50 USDC welcome bonus — no deposit required. To add more collateral during beta, top up with on-chain USDC, USDT, or SOL through the deposit flow in the trading terminal.

Beta notice

Demo balances are for testing only. They are not withdrawable as real USDC. On-chain top-ups are credited to your demo balance after transaction verification.

Developer setup

To run the Perpex app locally from GitHub:

git clone https://github.com/tradeperpex/tradeperpex.git
cd tradeperpex
pnpm install
cp .env.example .env.local
node scripts/migrate.mjs
pnpm dev

The app boots at http://localhost:3000. At minimum, configure DATABASE_URL, BIRDEYE_API_KEY, and PAYMENT_ADDRESS.
