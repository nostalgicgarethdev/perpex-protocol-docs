API reference Beta

Internal API routes used by the Perpex app. A public REST + WebSocket surface is planned for v0.3.

Base URL

https://tradeperpex.fun/api

Positions

MethodRouteDescription
POST/positionsOpen a position. Quotes mark from engine, writes row.
GET/positions?wallet=…List positions for a wallet.
POST/positions/closeClose position, settle PnL into demo balance.
POST/positions/update-tpslAttach or change TP/SL levels.
POST/positions/liquidateServer-side liquidator hook (engine callback).

Orders

MethodRouteDescription
POST/ordersPlace limit or stop-market order.
POST/orders/cancelCancel a resting order.
POST/orders/triggerTrigger a resting order (engine callback).

Markets

MethodRouteDescription
GET/marketsList user-created markets.
POST/marketsCreate a market (verified launch fee).
GET/markets/[symbol]Single-market detail.

Balances & deposits

MethodRouteDescription
GET/demo-balanceRead or seed welcome bonus.
POST/demo-balanceApply atomic delta (credit/debit).
POST/demo-depositVerify on-chain payment, credit demo USDC.
GET/verify-paymentIdempotent payment-status check.
GET/wallet-balanceLive SOL/USDC balance for an address.

Market data

MethodRouteDescription
GET/ohlcvProxied candles for the chart.
GET/tokensDiscover and search tokens.
GET/search-tokenSearch by mint or symbol.
GET/token-priceSingle price quote.
