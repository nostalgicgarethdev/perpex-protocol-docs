Resources

Report an issue

Found a bug, security vulnerability, or something broken? Here is how to reach the Perpex team.

Security disclosures

Security vulnerabilities should be reported privately. Do not file security issues on public GitHub.

Private disclosure only

Email security@tradeperpex.fun with a detailed description, reproduction steps, and impact assessment. We aim to respond within 48 hours.

Bug reports

For non-security bugs, open an issue on GitHub:

  1. Visit github.com/tradeperpex/tradeperpex/issues
  2. Choose the bug report template
  3. Include wallet address (if relevant), market symbol, steps to reproduce, and screenshots

Feature requests

Use the feature request template on GitHub Issues or reach out on @tradeperpex.

Incident status

Incidents and degradations are posted to @tradeperpex in real time. A dedicated status page is planned for v0.2.
